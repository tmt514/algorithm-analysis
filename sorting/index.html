排序 Sorting

排序 Sorting

排序是整理資料中一種最直接的方式。

演算法 時間複雜度 空間複雜度
泡沫排序法 最差 O(n2)O(n^2)O(n2), O(n+nτ)O(n + n\tau)O(n+nτ) 最差 O(1)O(1)O(1)
雞尾酒排序法 最差 O(n2)O(n^2)O(n2), O(n+nτ)O(n + n\sqrt{\tau})O(n+nτ​) 最差 O(1)O(1)O(1)
插入排序法 最差 O(n2)O(n^2)O(n2), O(n+τ)O(n + \tau)O(n+τ) 最差 O(1)O(1)O(1)
選擇排序法 最差 O(n2)O(n^2)O(n2) 最差 O(1)O(1)O(1)
合併排序法 最差 O(nlog⁡n)O(n\log n)O(nlogn) 最差 O(n)O(n)O(n)
原地合併排序 最差 O(nlog⁡n)O(n\log n)O(nlogn) 最差 O(1)O(1)O(1)
快速排序法 - 第一個數字 pivot 最差 O(n2)O(n^2)O(n2) 最差 O(d)=O(n)O(d) = O(n)O(d)=O(n)
快速排序法 - 隨機選 pivot 期望 O(nlog⁡n)O(n\log n)O(nlogn) 期望 O(d)=O(log⁡n)O(d) = O(\log n)O(d)=O(logn)
快速排序法 - 找中位數 pivot 最差 O(nlog⁡n)O(n\log n)O(nlogn) 最差 O(d)=O(log⁡n)O(d) = O(\log n)O(d)=O(logn)
原地快速排序 最差 O(n2)O(n^2)O(n2) 最差 O(1)O(1)O(1)
堆積排序法 最差 O(nlog⁡n)O(n\log n)O(nlogn) 最差 O(1)O(1)O(1)

註記

  • nnn 代表輸入的資料量（序列元素個數）。
  • τ\tauτ 代表輸入序列的逆序數對個數。
  • ddd 代表遞迴深度。
